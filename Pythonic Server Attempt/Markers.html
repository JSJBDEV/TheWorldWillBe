Instructions:

  1. Write a number in the first box, this is the seed, it can be any Integer (-1,0,1,2 etc)
  2. Write a number in the second box, this is the amount of years to generate so must be a natural (1,2,3,4,5 etc) number (high numbers take exponentionally longer to load/li>
  3. Press "Load Simulation File" wait for a message to display sayong "file loaded"
  4. When it has loaded you can either single step the simulation through years, or run the simulation until the amount of years specified
  5. to examine towns in a year click "Load Year To Examine" then "Examine Year" (the latter cleans up the map) when it has loaded you should be able to click on a town, then on the towns name; stats should be displayed
